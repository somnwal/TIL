### 이클립스에서 Maven Build

1. 프로젝트 혹은 pom.xml 우클릭
2. Run As > Maven Build
3. Goals 항목에 package 입력 후 Run
4. 생성된 .jar 파일 확인

&nbsp;
&nbsp;

### 스프링부트 프로젝트 실행
```bash
java -jar spingboot_project.jar
```

&nbsp;
&nbsp;

### 스프링부트 프로젝트 백그라운드로 실행
```bash
nohup java -jar spingboot_project.jar &
```
